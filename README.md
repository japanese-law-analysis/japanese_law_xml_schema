## 概要
[法令標準XMLスキーマ](https://elaws.e-gov.go.jp/file/XMLSchemaForJapaneseLaw_v3.xsd)で定義された規格に基づいてXMLとの相互変換を行うライブラリです。

テキストを解析して構造を構築することもできます。
